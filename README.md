##  Project Structure

The portfolio is organized into the following tasks:

###  Task 1 – Digital Literacy Awareness  
Created an infographic using Canva to explain digital literacy concepts such as safe internet practices, password security, and professional communication.

###  Task 2 – Digital Portfolio  
Built my professional presence on GitHub, LinkedIn, and Kaggle. GitHub is used to showcase projects, LinkedIn for networking, and Kaggle for data science practice.

###  Task 3 – Coding & Collaboration  
Practiced coding on HackerRank by solving beginner problems. Also created a Digital Literacy Quiz using Google Forms and connected it to Google Sheets for response analysis.

###  Task 4 – Professional Email & Etiquette  
Drafted formal emails for academic and professional use. Created a social media Do’s and Don’ts checklist to promote responsible online behavior.

###  Task 5 – Cybercrime Awareness  
Conducted a case study on phishing scams targeting students. Learned prevention techniques and safe online practices to avoid cyber fraud.
